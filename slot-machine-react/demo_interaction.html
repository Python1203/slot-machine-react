2️⃣ Canvas 渲染 vs DOM 渲染

❌ DOM 方案

每个符号一个 div 元素

<div class="reel"> <div>🍒</div> <div>💎</div> <div>7️⃣</div> </div>

缺点：

  • 15 个节点需要频繁更新
  • 浏览器重排重绘压力大
  • FPS: 30-40（卡顿）
✅ Canvas 方案

整个卷轴一个 canvas 元素

<canvas id="reel"></canvas> <script> const ctx = canvas.getContext('2d'); ctx.drawImage(texture, 0, y); </script>

优势：

  • GPU 加速渲染
  • 批量绘制效率高
  • FPS: 60（丝滑）

3️⃣ 缓动曲线（Easing Functions）

模拟真实物理的停止效果（Back-Out 回弹）

❌ 线性停止

position = start + change × progress

效果：机械式急停，不自然

✅ Back-Out 缓动

position = start + change × easeOutBack(progress)

效果：先冲过头再回弹，真实物理感

// Back-Out 缓动函数 easeOutBack(t) { const c1 = 1.70158; const c3 = c1 + 1; return 1 + c3 * Math.pow(t - 1, 3) + c1 * Math.pow(t - 1, 2); } // 应用示例 const progress = elapsedTime / duration; const easedProgress = easeOutBack(progress); currentPosition = startPosition + change * easedProgress;

📊 关键性能指标

0ms 输入到动画延迟
60 FPS (Canvas)
<100ms 网络请求超时
3 次 自动重试次数
